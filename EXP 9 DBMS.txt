🔹 THEORY (WITH SYNTAX)
✅ VIEWS
In SQL, a view is a virtual table based on the result of an SQL query. It does not store data physically but displays data from one or more tables. Views help simplify complex queries and enhance security by restricting access to specific data.
A view contains rows and columns like a real table, and it can include SQL functions, WHERE conditions, and JOIN operations.


🔸 1. CREATE VIEW
Syntax:
CREATE VIEW view_name AS
SELECT column_name(s)
FROM table_name
WHERE condition;


🔸 2. CREATE OR REPLACE VIEW
Used to modify an existing view without dropping it.
Syntax:
CREATE OR REPLACE VIEW view_name AS
SELECT column_name(s)
FROM table_name
WHERE condition;


🔸 3. DROP VIEW
Used to delete a view.
Syntax:
DROP VIEW view_name;


🔹 DATABASE TRIGGERS
Triggers are stored programs that automatically execute (fire) when specific events occur in the database. These events can be:
* DML operations (INSERT, UPDATE, DELETE)
* DDL operations (CREATE, ALTER, DROP)
* Database events (LOGON, LOGOFF, etc.)
Triggers can be defined on tables, views, schemas, or the entire database.


🔸 CREATE TRIGGER
Syntax:
CREATE OR REPLACE TRIGGER trigger_name
{BEFORE | AFTER | INSTEAD OF}
{INSERT OR UPDATE OR DELETE}
ON table_name
[FOR EACH ROW]
WHEN (condition)
DECLARE
-- variables
BEGIN
-- trigger logic
END;


🔸 Example (from file)
A trigger is created on the CUSTOMERS table to display salary changes when INSERT, UPDATE, or DELETE operations occur.
CREATE OR REPLACE TRIGGER display_salary_changes
BEFORE INSERT OR UPDATE OR DELETE ON customers
FOR EACH ROW
WHEN (NEW.ID > 0)
DECLARE
  sal_diff NUMBER;
BEGIN
  sal_diff := :NEW.salary - :OLD.salary;
  dbms_output.put_line('Old salary: ' || :OLD.salary);
  dbms_output.put_line('New salary: ' || :NEW.salary);
  dbms_output.put_line('Salary difference: ' || sal_diff);
END;
👉 As shown on page 3–4, this trigger fires automatically and displays salary differences when records are inserted or updated in the table.
